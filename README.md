# LodgeDesk

A simple property management system for small independent motels. See [PRD.md](PRD.md) for the full spec — it's the source of truth for this project.

## Stack

- Next.js (App Router) + TypeScript
- Tailwind CSS
- Prisma + PostgreSQL

## Getting started

1. Copy `.env.example` to `.env` and fill in a real `DATABASE_URL`.
2. Install dependencies: `npm install`
3. Run the dev server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)
